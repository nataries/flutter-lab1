## Контрольные вопросы

1. Чем final отличается от const в Dart?

final - неизменяемая переменная, значение которой присваивается один раз и известно во время выполнения.

const - используется для неизменяемых объектов, значение известно на этапе компиляции

2. Что означает String?

String - значение гарантированно не null

4. Чем Future отличается от обычного значения? Что означает await с точки зрения 
потока выполнения?

Future - это объект, который не содержит результат в данный момент, но получит его позже.

await приостанавливает выполнение текущей функции и не блокирует поток.

5. Зачем в Dart именованные конструкторы, если в C# есть перегрузка?

В Dart нет перегрузки, поэтому используются именованные конструкторы
